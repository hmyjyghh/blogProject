**git常用命令**

## 如何将本地项目上传到Github

### 常用命令


```
git add .

git commit -m "提交信息"

git push -u origin main
```


## 多人协作开发

### 常用命令

**拉取当前分支最新代码**

```
git pull  
```

**push到远程的master分支上**

```
git push origin master
```
