## 常用命令

### Create a new post

``` bash
hexo new page "about"

hexo new page categories

hexo new page contact

hexo new post "article title"
```
**hexo g 生成静态网页      hexo s 预览**

**然后运行hexo clean清空所有生成的网页缓存，hexo d 上传到github上，更新自己的博客**
